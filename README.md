## 项目介绍
基于jsp+servlet的超市账单管理系统：前端 jsp，后端 servlet，角色分为管理员、普通用户；集成账单管理、供应商管理、用户管理等功能于一体的系统。

## 功能介绍

基本功能：登录、退出
账单管理：账单列表查询，按商品名称搜索，按是否付款检索，添加账单数据，修改，删除
供应商管理：供应商列表查询，按名称搜索，按供应商描述搜索，添加供应商数据，修改，删除
用户管理：用户信息列表查询，按用户名称搜索，添加用户，查看详情，修改和删除
